# left4dead
TP Grupal de Taller de Programación

## Correr en 1920x1080

Para correr el juego:
```
mkdir build
cd build
cmake ..
make
./lesson00
```

Para correr los tests (cuando hayan):
```
mkdir build
cd build
cmake ..
make
./runTests
```
